Chat with Databricks

Conversational interface for governed Databricks SQL. Built with LangChain tool-calling, allowlisted views, PII redaction, and Delta audit trails.

Problem

Analysts need safe, conversational access to enterprise data without free-form SQL or credential risks.

Solution

  • Tool-calling to Databricks SQL endpoint with allowlist
  • Query templates and result-size guardrails
  • PII masking and Delta audit logging
