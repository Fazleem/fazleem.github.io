Case study

Chat with Databricks

Conversational interface for governed Databricks SQL. Built with LangChain tool-calling, allowlisted views, PII redaction, and Delta audit trails.

Problem

Analysts need safe, conversational access to enterprise data without free-form SQL or credential risks.

Solution

  • Tool-calling to Databricks SQL endpoint with allowlist
  • Query templates and result-size guardrails
  • PII masking and Delta audit logging

Architecture

01 Analyst prompt

Question enters a constrained assistant workflow rather than direct database access.

02 Tool guardrails

LangChain tool-calling maps the request to allowlisted views and query templates.

03 Databricks SQL

Queries run with result-size controls, PII masking, and credential separation.

04 Audit trail

Delta logging captures prompts, generated queries, responses, and review metadata.
